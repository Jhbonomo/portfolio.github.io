Designing for scale in Brazilian retail

Product design for Via's mobile e-commerce apps, improving the shopping experience for one of Brazil's largest retail companies, serving millions of Brazilian consumers across the country.
2021 Product Squad Capgemini/Via

Overview

Product designer for Via's mobile applications, a leading Brazilian retailer that owns three marketplace brands: Casas Bahia, Ponto, and Extra. I was part of the APP Experience Squad, improving mobile journeys for seamless user experiences alongside developers, QA, and project leads. I also contributed to Via's UX tribe, collaborating with designers to refine flows, share feedback, and join critique sessions.

The Challenge

Ahead of Black Friday, the app's UI underwent updates to align with the new design system guidelines. These included enhancements to text styles, color systems, and formatting, improving accessibility and delivering a more inclusive user experience.

My Role

  • Usability improvements: Improved mobile user journeys to facilitate the buying experience
  • Mobile-First Design: Intuitive mobile interfaces optimized for various screen sizes and capabilities
  • Accessibility Design: Ensured the platform was accessible to users with varying digital literacy
  • Design System implementation: helped to properly implement the new Design system adapting the design components that could work across the three white label applications

Featured categories

We enhanced navigation by adding quick-access cards to the homepage, spotlighting popular categories like mobile devices and electronics. This made it easier for users to locate key items, ultimately boosting interaction and improving user engagement during peak sales.

Login and onboarding journeys

The login process was a major challenge, with high bounce rates disrupting key user flows, particularly during the purchasing journey. To address this, we optimized the biometric login page, streamlining the experience to minimize friction. These improvements significantly reduced bounce rates, positively impacting the buying journey and reflected in enhanced user metrics.

Setting the delivery address

Our squad tackled features cutting across multiple user journeys, like managing user address and location details.

The Impact

I created monitoring dashboards in Looker Studio to track squad-implemented changes and deliver actionable insights to stakeholders. For example, I created a dashboard to report the login and onboarding processes, directly tying user metrics to the squad's enhancements.

Thank you for reading this case study. This project demonstrated how thoughtful mobile design can scale across diverse markets and user segments, creating experiences that serve millions of users while maintaining quality and accessibility. The insights and methodologies developed continue to influence mobile design approaches in complex, diverse markets.
